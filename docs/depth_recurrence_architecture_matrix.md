## Source Idea Map

- MoDA / Mixture-of-Depths Attention: attention heads can retrieve from current
  sequence KV and prior-depth KV, addressing depth signal dilution. Phase 1 uses
  a PyTorch approximation, not the paper's Flash-compatible kernel.
- Mixture-of-Depths compute routing: token-level top-k block routing is useful
  but separate from depth-KV retrieval. The current D1 prototype uses causal
  prefix-top-k routing to preserve autoregressive behavior.
- Recurrent-depth latent compute: loop a small middle block group and treat
  extra passes as latent refinement.
- Two-scale recurrent dynamics: cheap halting should use hidden-state dynamics,
  especially second-order changes in step size, rather than decoding logits.
- Looped transformers as programmable computers: keep only the practical lesson
  for tranche 1: shallow repeated blocks plus a compact scratchpad/expert path.
- Native sparse attention: valuable later for long-context efficiency, not part
  of this first small-model tranche.
