LLM Architecture Views

Seven architecture pages to compare standard transformers, hybrids, our original recursive attempt, the new recursive-memory direction, a speculative native internal-search runtime, the current `v3a` hybrid-attention structure, and a thought-channel hybrid for native internal search. The goal is not photorealism. The goal is a better mental model.

Native Internal Search

A Mermaid-rendered sketch of a hybrid search architecture that combines fractal state-space memory, selective attention, bounded latent hypotheses, and internal verify-or-halt loops.

Page 6

v3a Hybrid Attention

A Mermaid-rendered view of the current Path 1 build: interleaved local exact-attention layers plus either a Rust Mamba reference lane or our `P2` contender in the sequence-mixer slots.

Thought-Channel Hybrid

A Mermaid-rendered block diagram for a native-search model with a shared attention trunk, multiple fully active thought channels, and explicit compare-prune-merge control.

Recommended use: open the pages side by side and compare the same memory or retrieval story across architectures. On the 3D pages, flip runtime modes while doing that. That tends to make the cost story click much faster than reading prose alone.
